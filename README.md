# Rubyst

Rustで作るRuby

An Ruby made from Rust.

## 対応している文法

|型||
| :---:  | :---:|
| bool型 | Bool | 
| int型  | Int  |

|演算子||
| :---:  | :---:|
| 掛け算 | + | 
| 足し算 | - |
| 掛け算 | * |
| 割り算 | / |
| 余り | % |
| 乗算 | ** |

|比較演算子||
| :---:  | :---:|
|イコール|==| 
|大なりイコール|>=|
|小なりイコール|<=|
|大なり|>|
|小なり|<|

|関数||
| :---: | :---:|
| print文 | p() |
| if文 | if ~ end |

変数

## 記述例

```main.eld
(x = 1)
y = x + 1

if y == 2
  p(x + 2)
end

if y < 1
  p(x * 3)
end

p(y % 6)
```
